# Equidae reference boards

Keep one generated 2×2 studio board at `<breed-id>/board.(png|jpg|webp)`.
Use a neutral sage-gray seamless background (same as dog/cat/rodent stills).
Quadrants in reading order:

1. standing three-quarter
2. standing profile
3. front-facing standing conformation (`front-sit` slot — horses do not sit like dogs)
4. head close-up

One consistent adult animal, soft studio lighting, natural conformation, no
tack/bridle/saddle, no text, no props. Run `npm run prepare:equid-ref` to write
optimized scene JPEGs under `public/assets/equid-ref/<breed-id>/`.
